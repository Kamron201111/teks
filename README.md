Baza yo'q — har qidirishda API dan real vaqt ma'lumot oladi.

|-----------|----------------------------------|
| BOT_TOKEN | BotFather dan token              |
| ADMIN_ID  | Telegram ID (ixtiyoriy)          |

2. Bot API dan barcha sahifalarni aylanib qidiradi
3. Topilsa — telefon + sana ko'rsatadi
4. Topilmasa — ovoz berilmagan deydi
